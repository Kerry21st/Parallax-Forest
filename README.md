# Parallax Forest

Эффектный одностраничный сайт с параллакс-эффектом, создающий иллюзию глубины лесного пейзажа. При прокрутке страницы слои изображений движутся с разной скоростью, формируя объемную картинку.

## О проекте

Parallax Forest — это демонстрационный проект, показывающий возможности создания параллакс-эффектов с помощью современных веб-технологий. Главная особенность — многослойная анимация, где каждый слой (небо, лес, передний план) смещается относительно пользовательского скролла, создавая ощущение трехмерного пространства.

## Функциональность

- **Параллакс-эффект** — многослойная анимация с разной скоростью движения слоев
- **Адаптивный дизайн** — корректное отображение на экранах любого размера
- **Плавные анимации** — использование GSAP для высокопроизводительных переходов
- **Контроль скролла** — интеграция ScrollTrigger для синхронизации анимаций с прокруткой
- **Кастомные шрифты** — подключение шрифта Raleway для уникального стиля
- **Текстовые эффекты** — тени и свечение для заголовков

## Используемые технологии

- **HTML5** — семантическая структура
- **CSS3** — переменные, трансформации, кастомные свойства
- **GSAP (GreenSock Animation Platform)** — профессиональная библиотека для создания анимаций
- **ScrollTrigger** — плагин GSAP для привязки анимаций к прокрутке
- **ScrollSmoother** — плагин GSAP для плавного скролла

## Как это работает

1. **Многослойная структура** — страница разделена на несколько слоев (base, middle, front), каждый из которых позиционируется абсолютно
2. **Отслеживание скролла** — ScrollTrigger вычисляет позицию прокрутки и сохраняет ее в CSS-переменную `--scrollTop`
3. **Динамическое смещение** — каждый слой смещается с разным коэффициентом, создавая эффект глубины:
   - Базовый слой: `calc(var(--scrollTop) / 1.6)`
   - Средний слой: `calc(var(--scrollTop) / 2.5)`
   - Передний слой: `calc(var(--scrollTop) / 5.7)`
4. **Плавность** — ScrollSmoother обеспечивает плавную прокрутку без рывков

## Установка и запуск

1. Клонируйте репозиторий:
```
git clone https://github.com/Kerry21st/parallax-forest.git
```
Перейдите в папку проекта:

```
cd parallax-forest
```
Откройте файл index.html в любом современном браузере.
